When SN2 is performed on sterically hindered substrates, reaction is very slow and major reaction products are not from the substitution.
An elimination may take place instead.

For example, acetate may act as a base instead of a nucleophile when the substrate is sterically hindered, leading to elimination instead of substitution. 

Elimination is favoured by strong bases, sterically hindered substrates and high reaction temperatures.

Bimolecular elimination reactions, rate depends on both base and substrate, second order - E2

An E2 has to happen in an "anti-periplanar manner" (?)

A more alkylated alkene is more stable. 

For SN1, if no nucleophile is present to for the carbocation to interact with, a proton is expelled. Would normally happen in acidic conditions and non-aqueous solutions. Non-nucleophilic conditions are required, and so sulfuric acid would be used instead of hydrochloric due to hydrogen sulfate being less nucleophilic than chloride.

Reaction only depends on substrate concentration (unimolecular), first order - E1
As SN1 involves planar carbocations, racemization occurs in E1 as well.

Factors affecting substitution reactions:

|                 | SN1                    | SN2                    |
| --------------- | ---------------------- | ---------------------- |
| Substrate       | 3>2                    | CH3X>1>2               |
| Leaving group   | Good leaving groups    | Good leaving groups    |
| Nucleophile     | Small dependence of Nu | Large dependence of Nu |
| Kinetics        | First order            | Second order           |
| Stereochemistry | Racemization           | Inversion              |
| Rearrangements  | Common if favoured     | Not observed           |

Factors affecting elimination reactions:

|                 | E1                              | E2                              |
| --------------- | ------------------------------- | ------------------------------- |
| Substrate       | 3>2                             | 3>2                             |
| Leaving group   | Good leaving groups             | Good leaving groups             |
| Nucleophile     | Small dependence of base        | Large dependence of base        |
| Kinetics        | First order                     | Second order                    |
| Stereochemistry | No preferred geometry           | Antiperiplanar transition state |
| Rearrangements  | Common if favoured              | Not observed                    |
| Orientation     | Most alkylated alkene dominates | Most alkylated alkene dominates |

When carbocations appears as intermediates it is important to consider:
- Carbocations are planar sp2 - racemization
- Stability order: resonance stabilized > 3 > 2 > 1
- 1 will not normally form
- Carbocations will rearrange if a more stable cation can be formed
- If a nucleophile is present, SN1 will be favoured
- E1 will occur in absence of nucleophiles
